New Hire Paperwork

This portal can be used to submit New Hire paperwork to HR. Once approved, the new hire will obtain an email with the username nad password to login. You, as the hiring manager, will also obtain an email with the same login information as a confirmation they have been approved and hired.

Incident Report

This portal can be used to submit Incident Reports to management. This report should be used to document (potentially) serious incidents that need to be tracked. This form can be submitted even if a write up is not issued. Do not submit petty incidents using this form.

Employee Termination

This portal can be used to terminate employees who no longer work for the company. This will render their Microsoft Office 365 account unusable. This means all access to files, Teams (including shifts), and email will be inaccessible.
